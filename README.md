# ONCE-OS

**ONCE-OS** es un sistema de productividad desarrollado como aplicación web, diseñado para organizar tareas, mejorar la concentración y ofrecer una experiencia clara y eficiente tanto en computadora como en dispositivos móviles.

El proyecto forma parte del ecosistema **ONCE CORE**, enfocado en crear herramientas simples pero poderosas para la gestión personal y académica.

---

## Características Principales

* **Sistema de gestión de tareas**

  * Crear, visualizar y administrar tareas fácilmente.
  * Organización clara para mejorar el flujo de trabajo.

* **Temporizador Pomodoro integrado**

  * Reloj de concentración basado en la técnica Pomodoro.
  * Ideal para sesiones de estudio o programación.

* **Nuevo estilo visual**

  * Interfaz más limpia y estética.
  * Mejora de colores y legibilidad para uso prolongado.

* **Compatible con teléfonos**

  * Diseño adaptado para navegadores móviles.
  * Permite gestionar tareas y usar el Pomodoro desde cualquier lugar.

* **Interfaz simple y rápida**

  * Pensado para abrir, usar y concentrarse sin distracciones.

---

## Tecnologías Utilizadas

* HTML
* CSS
* JavaScript
* Claudflare (backend y almacenamiento de datos)

---

## Objetivo del Proyecto

El objetivo de **ONCE-OS** es construir un sistema ligero pero funcional para mejorar la productividad personal y académica, manteniendo una arquitectura clara y fácil de expandir.

Este proyecto también sirve como laboratorio de desarrollo para explorar:

* diseño de interfaces
* arquitectura de aplicaciones web
* sistemas de productividad
* integración con backend
